# Airspace Monitoring System

Учебный проект для мониторинга самолетов с использованием API.

## Описание проекта

Приложение получает информацию о самолетах,
находящихся в воздушном пространстве выбранной страны.

Данные получаются с использованием:
- OpenStreetMap Nominatim API
- OpenSky Network API

Проект позволяет:
- получать самолеты по стране
- сортировать самолеты по высоте
- получать Top N самолетов
- фильтровать самолеты по стране регистрации
- сохранять данные в JSON и CSV
- рассчитывать статистику

## Установка проекта:

Клонирование репозитория:
```
git clone https://github.com/almsar99/airspace-monitoring-system.git
```
### Переход в папку проекта:
```
cd airspace-monitoring-system
```
### Создание виртуального окружения:
```
python3 -m venv venv
```
### Активация виртуального окружения:
```
source venv/bin/activate
```
### Установка зависимостей:
```
pip install -r requirements.txt
```
## Запуск проекта:
```
python src/main.py
```
